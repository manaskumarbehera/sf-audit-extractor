üíæ Cache Manager New in v1.1.14

The Cache Manager provides intelligent caching of metadata and frequently used data to improve extension performance.

Features

  • Automatic Caching - Object and field metadata is automatically cached
  • Smart Invalidation - Cache is intelligently refreshed when needed
  • Performance Boost - Faster load times for objects and fields
  • Storage Management - Efficient use of browser storage
üí°
Tip: Clear Cache If you notice stale data, you can clear the cache from Settings ‚Üí Clear Cache.

üî¨ Record Scanner

The Record Scanner is a comprehensive tool for exploring Salesforce records.

Features

  • Record Details - View object type, ID, name, created/modified info
  • Field History - See tracked field changes with old ‚Üí new values
  • Related Records - Explore child relationships with counts
  • Recent Records - Quick access to last 5 scanned records
  • Dev Tools - Quick links to Setup, Console, Object Manager

How to Use

  1. Navigate to any record page in Salesforce
  2. Open TrackForcePro extension
  3. Go to Explore ‚Üí Record Scanner
  4. Record details, field history, and related records load automatically
  5. Or enter any 15/18 character ID to scan a different record

  • Close (√ó) - Hides the blinker for current session

üîß Troubleshooting

Unable to connect to Salesforce

Cause Solution
Content script not loaded Refresh the Salesforce page, then reopen the extension
Not logged in Log into Salesforce in the active tab
Extension updated Reload the extension from chrome://extensions
Session expired Log out and log back into Salesforce

Field History not loading

  • Ensure history tracking is enabled for the object in Salesforce Setup
  • Check that you have permission to view the history object
  • Some objects (User, Profile) don't support field history

Cache issues

  • Go to Settings ‚Üí Clear Cache to refresh cached data
  • Reload the Salesforce page after clearing cache
