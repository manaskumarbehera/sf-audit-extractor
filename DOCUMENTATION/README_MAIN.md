## üîß What Is TrackForcePro?

TrackForcePro is a Chrome extension for Salesforce that provides:

- **Audit Data Extraction** - Export Salesforce audit logs
- **SOQL Query Builder** - Build and execute SOQL queries
- **GraphQL Support** - Query using GraphQL
- **Platform Management** - Manage platform settings
- **LMS Support** - Lightning Message Service publishing
- **Data Explorer** - Browse and explore data
- **Multi-Org Support** - Work with multiple Salesforce orgs
